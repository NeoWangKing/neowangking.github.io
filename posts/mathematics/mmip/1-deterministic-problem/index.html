数学物理方程的导出

波动方程

波动方程的常见形式可以写成

utt−a2Δu=0\begin{align} u_{tt} - a^{2}\Delta u = 0 \end{align}utt​−a2Δu=0​​

  1. 均匀弦的微小横振动

    对应 a2=FT/ρa^{2} = F_{T} / \rhoa2=FT​/ρ ，方程为

    ρutt−FTuxx=0\begin{align} \rho u_{tt} - F_{T}u_{xx} = 0 \end{align}ρutt​−FT​uxx​=0​​

    即为弦振动方程

    考虑受迫振动的情形，化为

    ρutt−FTuxx=F(x,t)⇒utt−a2uxx=f(x,t)\begin{align} \rho u_{tt} - F_{T}u_{xx} &= F(x,t)\\ \Rightarrow u_{tt} - a^{2}u_{xx} &= f(x,t) \end{align}ρutt​−FT​uxx​⇒utt​−a2uxx​​=F(x,t)=f(x,t)​​

    就是弦的受迫振动方程

  2. 均匀杆的纵振动

    对应 a2=E/ρa^{2} = E / \rhoa2=E/ρ ，方程为

    ρutt−Euxx=0\begin{align} \rho u_{tt} - Eu_{xx} = 0 \end{align}ρutt​−Euxx​=0​​

    即为杆的纵振动方程

    考虑受迫振动的情形，化为

    ρutt−Euxx=F(x,t)⇒utt−a2uxx=f(x,t)\begin{align} \rho u_{tt} - Eu_{xx} &= F(x,t)\\ \Rightarrow u_{tt} - a^{2}u_{xx} &= f(x,t) \end{align}ρutt​−Euxx​⇒utt​−a2uxx​​=F(x,t)=f(x,t)​​

    就是杆的受迫振动方程

  3. 传输线方程（电报方程）

    1. 非理想传输线

      方程为

      LCjtt−jxx+(LG+RC)jt+RGj=0LCvtt−vxx+(LG+RC)vt+RGv=0\begin{align} LCj_{tt} - j_{xx} + (LG+RC)j_{t} + RGj &= 0\\ LCv_{tt} - v_{xx} + (LG+RC)v_{t} + RGv &= 0 \end{align}LCjtt​−jxx​+(LG+RC)jt​+RGjLCvtt​−vxx​+(LG+RC)vt​+RGv​=0=0​​

    2. 理想传输线

      在导线电阻 RRR 和线间电导 GGG 很小以至于可以忽略时，化成理想传输线

      jtt−a2jxx=0vtt−a2vxx=0\begin{align} j_{tt} - a^{2}j_{xx} &= 0\\ v_{tt} - a^{2}v_{xx} &= 0 \end{align}jtt​−a2jxx​vtt​−a2vxx​​=0=0​​

      其中 a2=1/LCa^{2} = 1 / LCa2=1/LC

    上述四个方程叫做传输线方程（电报方程）

  4. 均匀薄膜的微小横振动

    方程

    ρutt−FT(uxx+uyy)=0\begin{align} \rho u_{tt} - F_{T}(u_{xx} + u_{yy}) = 0 \end{align}ρutt​−FT​(uxx​+uyy​)=0​​

    用拉普拉斯算符简化，即可得到

    ρutt−FTΔ2u=0utt−a2Δ2u=0\begin{align} \rho u_{tt} - F_{T}\Delta_{2}u &= 0\\ u_{tt} - a^{2}\Delta_{2}u &= 0 \end{align}ρutt​−FT​Δ2​uutt​−a2Δ2​u​=0=0​​

    这就是薄膜微小振动方程

    在受迫振动的情形下

    utt−a2Δ2u=f(x,y,t)\begin{align} u_{tt} - a^{2}\Delta_{2}u = f(x,y,t) \end{align}utt​−a2Δ2​u=f(x,y,t)​​

    就是薄膜的受迫振动方程

  5. 流体力学与声学方程

    对应 a2=γp0ρ0a^{2}=\frac{\gamma p_{0}}{\rho_{0}}a2=ρ0​γp0​​ ，方程为

    utt−a2Δ3u=0\begin{align} u_{tt} - a^{2}\Delta_{3}u = 0 \end{align}utt​−a2Δ3​u=0​​

    就是声波方程，其中 p0p_{0}p0​ 和 ρ0\rho_{0}ρ0​ 为空气处于平衡状态下的压强和密度

  6. 电磁波方程

    利用麦克斯韦方程组可以推导出

    Ett−a2Δ3E=0Htt−a2Δ3H=0\begin{align} \boldsymbol{E}_{tt} - a^{2}\Delta_{3}\boldsymbol{E} &= 0\\ \boldsymbol{H}_{tt} - a^{2}\Delta_{3}\boldsymbol{H} &= 0 \end{align}Ett​−a2Δ3​EHtt​−a2Δ3​H​=0=0​​

    此即为真空中的电磁波方程，其中 a2=1/μ0ε0=光速平方a^{2} = 1 / \mu_{0}\varepsilon_{0} = \text{光速平方}a2=1/μ0​ε0​=光速平方

  7. 杆的微小横振动

    杆在横向变形时，存在切应力，于是杆的横振动方程为

    utt+a2uxxxx=0(a2=EI/ρ)\begin{align} u_{tt} + a^{2}u_{xxxx} = 0\quad(a^{2} = EI / \rho) \end{align}utt​+a2uxxxx​=0(a2=EI/ρ)​​

    其中 EEE 是弹性模量，III 是转动惯量，ρ\rhoρ 是密度

    注意这个方程与前面的振动的常用方程的不同，比如符号、偏导的阶数

    ut−[∂∂x(D∂u∂x)+∂∂y(D∂u∂y)+∂∂z(D∂u∂z)]=0\begin{align} u_{t} - \left[\frac{\partial}{\partial x}\left(D\frac{\partial u}{\partial x}\right) + \frac{\partial}{\partial y}\left(D\frac{\partial u}{\partial y}\right) + \frac{\partial}{\partial z}\left(D\frac{\partial u}{\partial z}\right)\right] = 0 \end{align}ut​−[∂x∂​(D∂x∂u​)+∂y∂​(D∂y∂u​)+∂z∂​(D∂z∂u​)]=0​​

  2. 热传导方程

    热传导的起源是温度的不均匀，热传导现象遵循热传导定律，也就是傅里叶定律

稳定场方程

  1. 稳定浓度分布

    如果扩散源的强度 F(x,y,z)F(x,y,z)F(x,y,z) 不随时间变化，扩散运动最后会达到稳定状态，于是方程 (22)\text{(22)}(22) 就化为

    ∇⋅(D∇u)=−F(x,y,z)\begin{align} \nabla \cdot (D\nabla u) = -F(x,y,z) \end{align}∇⋅(D∇u)=−F(x,y,z)​​

    如果 DDD 是常数，就有

    DΔu=−F(x,y,z)\begin{align} D\Delta u = -F(x,y,z) \end{align}DΔu=−F(x,y,z)​​

    这就是泊松方程，如果没有源，就是拉普拉斯方程

    Δu=0\begin{align} \Delta u = 0 \end{align}Δu=0​​

    这两个方程就是浓度的稳定分布方程

  2. 稳定温度分布

    与浓度分布对应，如果热源温度 T(x,y,z)T(x,y,z)T(x,y,z) 不随时间变化，方程 (28)\text{(28)}(28) 就化为

    ∇⋅(k∇u)=−T(x,y,z)\begin{align} \nabla \cdot (k\nabla u) = -T(x,y,z) \end{align}∇⋅(k∇u)=−T(x,y,z)​​

    这也是泊松方程，如果没有热源，也会简化为拉普拉斯方程

    Δu=0\begin{align} \Delta u = 0 \end{align}Δu=0​​

  3. 静电场

    静电场的电场强度 E\boldsymbol{E}E 是无旋的，即

    ∇×E=0\begin{align} \nabla \times \boldsymbol{E} = 0 \end{align}∇×E=0​​

    于是存在电势函数 V(x,y,z)V(x,y,z)V(x,y,z) 使得

    E=−∇V\begin{align} \boldsymbol{E} = -\nabla V \end{align}E=−∇V​​

    ∇⋅E=1ε0ρ\begin{align} \nabla \cdot \boldsymbol{E} = \frac{1}{\varepsilon_{0}}\rho \end{align}∇⋅E=ε0​1​ρ​​

    代入即可得到泊松方程

    ΔV=−1ε0ρ\begin{align} \Delta V = -\frac{1}{\varepsilon_{0}}\rho \end{align}ΔV=−ε0​1​ρ​​

    在没有电荷分布的区域内，方程简化为拉普拉斯方程

    ΔV=0\begin{align} \Delta V = 0 \end{align}ΔV=0​​

  4. 恒定电流场

    研究具有恒定电流分布的导电介质，其中的电荷满足连续性方程

    ∂ρ∂t+∇⋅j=0\begin{align} \frac{\partial \rho}{\partial t} + \nabla \cdot \boldsymbol{j} = 0 \end{align}∂t∂ρ​+∇⋅j=0​​

    在恒定电流的情况下，j(x,y,z)\boldsymbol{j}(x,y,z)j(x,y,z) 不随时间变化，ρ(x,y,z)\rho(x,y,z)ρ(x,y,z) 也不随时间变化，于是

    Δφ=0\begin{align} \Delta\varphi = 0 \end{align}Δφ=0​​

    这是拉普拉斯方程

定解条件

（一）初始条件

对于随着时间而发展变化的问题，必须考虑到研究对象的“历史”，即追溯到某个“初始时刻”的状态，也即初始条件

  1. 输运过程

    对于输运过程来说，初始状态指的是所研究物理量 uuu 的初始分布，因此初始条件就是

    u(x,y,z,t)∣t=0=φ(x,y,z)\begin{align} \left. u(x,y,z,t)\right|_{t = 0} = \varphi(x,y,z) \end{align}u(x,y,z,t)∣t=0​=φ(x,y,z)​​

  2. 振动过程

    对于振动过程，只给出物理量的初始“位置”

    u(x,y,z,t)∣t=0=φ(x,y,z)\begin{align} \left. u(x,y,z,t)\right|_{t = 0} = \varphi(x,y,z) \end{align}u(x,y,z,t)∣t=0​=φ(x,y,z)​​

    是不够的，还需要知道初始的“速度”

    ut(x,y,z,t)∣t=0=ψ(x,y,z)\begin{align} \left. u_{t}(x,y,z,t)\right|_{t = 0} = \psi(x,y,z) \end{align}ut​(x,y,z,t)∣t=0​=ψ(x,y,z)​​

    这是由于振动方程中出现了关于时间 ttt 的二阶导数，所以需要两个初始条件

  3. 稳定场问题不存在初始条件问题

（二）边界条件

研究具体的物理系统时，需要考虑研究对象所处的特定环境，这体现在边界上的物理状况，也就是边界条件

  1. 线性边界条件

    数学上，可以将常见的线性边界条件分为三类

    1. 第一类边界条件

      直接规定了所研究的物理量在边界上的数值

      u(r,t)∣Σ=f(M,t)\begin{align} \left. u(\boldsymbol{r},t)\right|_{\Sigma} = f(M,t) \end{align}u(r,t)∣Σ​=f(M,t)​​

    2. 第二类边界条件

      规定了物理量在边界外法线方向上方向导数的数值

      ∂u∂n∣Σ=f(M,t)\begin{align} \left. \frac{\partial u}{\partial n}\right|_{\Sigma} = f(M,t) \end{align}∂n∂u​​Σ​=f(M,t)​​

    3. 第三类边界条件

      规定了物理量及其外法向导数的线性组合在边界上的数值

      (u+H∂u∂n)∣Σ=f(M,t)\begin{align} \left. \left(u + H\frac{\partial u}{\partial n}\right)\right|_{\Sigma} = f(M,t) \end{align}(u+H∂n∂u​)​Σ​=f(M,t)​​

  2. 非线性边界条件

    比如物体表面按照斯蒂芬定律（与温度的四次方成正比）向四周辐射能量时，就会出现非线性的边界条件

（三）衔接条件

有时候在所研究的区域里面会出现跃变点，此时可以将整体进行分段考虑，并且结合衔接条件进行分析

    {A11=a11ξx2+2a12ξxξy+a22ξy2A12=a11ξxηx+a12(ξxηy+ξyηx)+a22ξyηyA22=a11ηx2+2a12ηxηy+a22ηy2B1=a11ξxx+2a12ξxy+a22ξyy+b1ξx+b2ξyB2=a11ηxx+2a12ηxy+a22ηyy+b1ηx+b2ηyC=cF=f\begin{align} \begin{cases} \begin{aligned} &A_{11} = a_{11}\xi_{x}^{2} + 2a_{12}\xi_{x}\xi_{y} + a_{22}\xi_{y}^{2}\\ &A_{12} = a_{11}\xi_{x}\eta_{x} + a_{12}(\xi_{x}\eta_{y} + \xi_{y}\eta_{x}) + a_{22}\xi_{y}\eta_{y}\\ &A_{22} = a_{11}\eta_{x}^{2} + 2a_{12}\eta_{x}\eta_{y} + a_{22}\eta_{y}^{2}\\ &B_{1} = a_{11}\xi_{xx} + 2a_{12}\xi_{xy} + a_{22}\xi_{yy} + b_{1}\xi_{x} + b_{2}\xi_{y}\\ &B_{2} = a_{11}\eta_{xx} + 2a_{12}\eta_{xy} + a_{22}\eta_{yy} + b_{1}\eta_{x} + b_{2}\eta_{y}\\ &C = c\\ &F = f \end{aligned} \end{cases} \end{align}⎩⎨⎧​​A11​=a11​ξx2​+2a12​ξx​ξy​+a22​ξy2​A12​=a11​ξx​ηx​+a12​(ξx​ηy​+ξy​ηx​)+a22​ξy​ηy​A22​=a11​ηx2​+2a12​ηx​ηy​+a22​ηy2​B1​=a11​ξxx​+2a12​ξxy​+a22​ξyy​+b1​ξx​+b2​ξy​B2​=a11​ηxx​+2a12​ηxy​+a22​ηyy​+b1​ηx​+b2​ηy​C=cF=f​​​​

    由此可见，如果选取一个满足方程

    a11ξx2+2a12ξxξy+a22ξy2=0\begin{align} a_{11}\xi_{x}^{2} + 2a_{12}\xi_{x}\xi_{y} + a_{22}\xi_{y}^{2} = 0 \end{align}a11​ξx2​+2a12​ξx​ξy​+a22​ξy2​=0​​

    的代换 ξ\xiξ，则就有 A11=0A_{11} = 0A11​=0，同理也可以选取合适的 η\etaη 使得 A22=0A_{22} = 0A22​=0，这样方程就可以得到化简

    进一步的，如果把 ξ(x,y)=常数\xi(x, y) = \text{常数}ξ(x,y)=常数 当作定义隐函数 y(x)y(x)y(x) 的方程，则 dy/dx=−ξx/ξy\mathrm{d}y / \mathrm{d}x = -\xi_{x} / \xi_{y}dy/dx=−ξx​/ξy​ 从而上述方程就是

    a11(dydx)2−2a12dydx+a22=0\begin{align} a_{11}\left(\frac{\mathrm{d}y}{\mathrm{d}x}\right)^{2} - 2a_{12}\frac{\mathrm{d}y}{\mathrm{d}x} + a_{22} = 0 \end{align}a11​(dxdy​)2−2a12​dxdy​+a22​=0​​

    这就是二阶线性偏微分方程的特征方程，特征方程的一般积分“ ξ(x,y)=常数\xi(x, y) = \text{常数}ξ(x,y)=常数 ”和“ η(x,y)=常数\eta(x, y) = \text{常数}η(x,y)=常数 ”叫做特征线

    特征方程可以分为两个方程

    dydx=a12+a122−a11a22a11dydx=a12−a122−a11a22a11\begin{align} \frac{\mathrm{d}y}{\mathrm{d}x} = \frac{a_{12} + \sqrt{a_{12}^{2} - a_{11}a_{22}}}{a_{11}}\\ \frac{\mathrm{d}y}{\mathrm{d}x} = \frac{a_{12} - \sqrt{a_{12}^{2} - a_{11}a_{22}}}{a_{11}} \end{align}dxdy​=a11​a12​+a122​−a11​a22​​​dxdy​=a11​a12​−a122​−a11​a22​​​​​

    由此可以划分出二元二阶线性偏微分方程 (54)(54)(54) 的类型

    {a122−a11a22>0,双曲型a122−a11a22=0,抛物型a122−a11a22<0,椭圆型\begin{align} \begin{cases} \begin{aligned} &a_{12}^{2} - a_{11}a_{22} > 0,\quad\text{双曲型}\\ &a_{12}^{2} - a_{11}a_{22} = 0,\quad\text{抛物型}\\ &a_{12}^{2} - a_{11}a_{22} < 0,\quad\text{椭圆型} \end{aligned} \end{cases} \end{align}⎩⎨⎧​​a122​−a11​a22​>0,双曲型a122​−a11​a22​=0,抛物型a122​−a11​a22​<0,椭圆型​​​​

    A122−A11A22=(a122−a11a22)(ξxηy−ξyηx)2\begin{align} A_{12}^{2}-A_{11}A_{22}=(a_{12}^{2}-a_{11}a_{22})(\xi_{x}\eta_{y}-\xi_{y}\eta_{x})^{2} \end{align}A122​−A11​A22​=(a122​−a11​a22​)(ξx​ηy​−ξy​ηx​)2​​
