留数定理

留数的定义

设 z0z_0z0​ 是解析函数 f(z)f(z)f(z) 的孤立奇点，在 z0z_0z0​ 的去心邻域内 f(z)f(z)f(z) 有洛朗展开

f(z)=∑n=−∞∞cn(z−z0)n\begin{align} f(z) = \sum_{n=-\infty}^{\infty} c_n (z - z_0)^n \end{align}f(z)=n=−∞∑∞​cn​(z−z0​)n​​

则 c−1c_{-1}c−1​ 称为 f(z)f(z)f(z) 在 z0z_0z0​ 处的 留数，记作 Res[f,z0]\mathrm{Res}[f, z_0]Res[f,z0​]。

由洛朗系数的积分公式，

Res[f,z0]=c−1=12πi∮Cf(z) dz,\begin{align} \mathrm{Res}[f, z_0] = c_{-1} = \frac{1}{2\pi \mathrm{i}} \oint_C f(z) \, dz, \end{align}Res[f,z0​]=c−1​=2πi1​∮C​f(z)dz,​​

其中 CCC 是环绕 z0z_0z0​ 一周且不包含其他奇点的简单闭曲线（逆时针方向）。

留数定理

定理：设 CCC 是一条正向简单闭曲线，f(z)f(z)f(z) 在 CCC 内部除有限个孤立奇点 z1,z2,…,znz_1, z_2, \dots, z_nz1​,z2​,…,zn​ 外处处解析，且在 CCC 上解析，则

∮Cf(z) dz=2πi∑k=1nRes[f,zk]\begin{align} \oint_C f(z) \, dz = 2\pi \mathrm{i} \sum_{k=1}^{n} \mathrm{Res}[f, z_k] \end{align}∮C​f(z)dz=2πik=1∑n​Res[f,zk​]​​

证明思路：将 CCC 内每个奇点用小圆周包围，利用复连通区域的柯西定理，将积分转化为各小圆周上积分的和，再由留数定义得证。

留数的计算

对于不同类型的奇点，留数有简便算法：

  • 可去奇点：c−1=0c_{-1}=0c−1​=0。

  • 一阶极点：若 z0z_0z0​ 为 f(z)f(z)f(z) 的一阶极点，则

    Res[f,z0]=lim⁡z→z0(z−z0)f(z)\begin{align} \mathrm{Res}[f, z_0] = \lim_{z\to z_0} (z - z_0) f(z) \end{align}Res[f,z0​]=z→z0​lim​(z−z0​)f(z)​​

    特别地，若 f(z)=P(z)Q(z)f(z) = \frac{P(z)}{Q(z)}f(z)=Q(z)P(z)​，P(z0)≠0P(z_0) \neq 0P(z0​)=0，Q(z0)=0Q(z_0)=0Q(z0​)=0，Q′(z0)≠0Q'(z_0) \neq 0Q′(z0​)=0，则

    Res[f,z0]=P(z0)Q′(z0)\begin{align} \mathrm{Res}[f, z_0] = \frac{P(z_0)}{Q'(z_0)} \end{align}Res[f,z0​]=Q′(z0​)P(z0​)​​​

  • mmm 阶极点：若 z0z_0z0​ 为 mmm 阶极点，则

    Res[f,z0]=1(m−1)!lim⁡z→z0dm−1dzm−1[(z−z0)mf(z)]\begin{align} \mathrm{Res}[f, z_0] = \frac{1}{(m-1)!} \lim_{z\to z_0} \frac{d^{m-1}}{dz^{m-1}} \left[ (z - z_0)^m f(z) \right] \end{align}Res[f,z0​]=(m−1)!1​z→z0​lim​dzm−1dm−1​[(z−z0​)mf(z)]​​

应用留数定理计算实变函数的定积分

留数定理的核心应用之一是计算某些实变函数的定积分，特别是广义积分。基本思路是将实积分转化为复平面上的围道积分，然后利用留数定理计算。常见的积分类型及处理方法如下。

类型一：三角函数有理式的积分

考虑形如 ∫02πR(cos⁡θ,sin⁡θ) dθ\int_{0}^{2\pi} R(\cos\theta, \sin\theta) \, d\theta∫02π​R(cosθ,sinθ)dθ 的积分，其中 RRR 是 cos⁡θ,sin⁡θ\cos\theta, \sin\thetacosθ,sinθ 的有理函数。令 z=eiθz = e^{\mathrm{i}\theta}z=eiθ，则 dθ=dzizd\theta = \frac{dz}{\mathrm{i}z}dθ=izdz​，cos⁡θ=z+z−12\cos\theta = \frac{z + z^{-1}}{2}cosθ=2z+z−1​，sin⁡θ=z−z−12i\sin\theta = \frac{z - z^{-1}}{2\mathrm{i}}sinθ=2iz−z−1​，积分变为单位圆上的围道积分：

∫02πR(cos⁡θ,sin⁡θ) dθ=∮∣z∣=1F(z) dziz,\begin{align} \int_{0}^{2\pi} R(\cos\theta, \sin\theta) \, d\theta = \oint_{|z|=1} F(z) \, \frac{dz}{\mathrm{i}z}, \end{align}∫02π​R(cosθ,sinθ)dθ=∮∣z∣=1​F(z)izdz​,​​

其中 F(z)F(z)F(z) 是 zzz 的有理函数。然后计算单位圆内所有奇点的留数和，乘以 2π2\pi2π 即得结果。

类型二：有理函数的无穷积分

考虑形如 ∫−∞∞R(x) dx\int_{-\infty}^{\infty} R(x) \, dx∫−∞∞​R(x)dx 的积分，其中 R(x)=P(x)/Q(x)R(x) = P(x)/Q(x)R(x)=P(x)/Q(x)，P,QP,QP,Q 为多项式，分母次数至少比分子高两次，且分母无实零点。则

∫−∞∞R(x) dx=2πi∑上半平面Res[R(z),zk],\begin{align} \int_{-\infty}^{\infty} R(x) \, dx = 2\pi \mathrm{i} \sum_{\text{上半平面}} \mathrm{Res}[R(z), z_k], \end{align}∫−∞∞​R(x)dx=2πi上半平面∑​Res[R(z),zk​],​​

其中求和取上半平面内的所有孤立奇点（通常是极点）。证明采用上半圆围道，利用大圆弧引理可得。

类型三：含三角函数的无穷积分

考虑形如 ∫−∞∞R(x)eiax dx\int_{-\infty}^{\infty} R(x) e^{\mathrm{i} a x} \, dx∫−∞∞​R(x)eiaxdx 的积分，其中 a>0a>0a>0，R(x)R(x)R(x) 为有理函数，分母次数高于分子次数，且分母无实零点。则

∫−∞∞R(x)eiax dx=2πi∑上半平面Res[R(z)eiaz,zk]\begin{align} \int_{-\infty}^{\infty} R(x) e^{\mathrm{i} a x} \, dx = 2\pi \mathrm{i} \sum_{\text{上半平面}} \mathrm{Res}\left[ R(z) e^{\mathrm{i} a z}, z_k \right] \end{align}∫−∞∞​R(x)eiaxdx=2πi上半平面∑​Res[R(z)eiaz,zk​]​​

取其实部或虚部可得 ∫R(x)cos⁡(ax)dx\int R(x) \cos(ax) dx∫R(x)cos(ax)dx 或 ∫R(x)sin⁡(ax)dx\int R(x) \sin(ax) dx∫R(x)sin(ax)dx。证明需用到约当引理（Jordan's Lemma）以确保上半圆弧积分趋于零。

类型四：实轴上有单极点的积分（柯西主值）

若 R(x)R(x)R(x) 在实轴上有单极点，则需取柯西主值，并以小半圆绕开极点。例如

P.V.∫−∞∞R(x)eiax dx=πi∑实轴上极点Res+2πi∑上半平面Res\begin{align} \mathrm{P.V.} \int_{-\infty}^{\infty} R(x) e^{\mathrm{i} a x} \, dx = \pi \mathrm{i} \sum_{\text{实轴上极点}} \mathrm{Res} + 2\pi \mathrm{i} \sum_{\text{上半平面}} \mathrm{Res} \end{align}P.V.∫−∞∞​R(x)eiaxdx=πi实轴上极点∑​Res+2πi上半平面∑​Res​​

计算定积分的一些例子

例1：计算一个三角函数有理式的积分（a>1a>1a>1）

计算 I=∫02πdθa+cos⁡θI = \int_{0}^{2\pi} \frac{d\theta}{a + \cos\theta}I=∫02π​a+cosθdθ​，其中 a>1a>1a>1。

解：令 z=eiθz = e^{\mathrm{i}\theta}z=eiθ，则 cos⁡θ=z+z−12\cos\theta = \frac{z+z^{-1}}{2}cosθ=2z+z−1​，dθ=dz/(iz)d\theta = dz/(\mathrm{i}z)dθ=dz/(iz)，积分化为

I=∮∣z∣=11a+z+z−12dziz=2i∮∣z∣=1dzz2+2az+1\begin{align} I = \oint_{|z|=1} \frac{1}{a + \frac{z+z^{-1}}{2}} \frac{dz}{\mathrm{i}z} = \frac{2}{\mathrm{i}} \oint_{|z|=1} \frac{dz}{z^2 + 2a z + 1} \end{align}I=∮∣z∣=1​a+2z+z−1​1​izdz​=i2​∮∣z∣=1​z2+2az+1dz​​​

分母 z2+2az+1=0z^2 + 2a z + 1 = 0z2+2az+1=0 的根为 z=−a±a2−1z = -a \pm \sqrt{a^2-1}z=−a±a2−1​。由于 a>1a>1a>1，两根皆为实数，且乘积为 111，故一根在单位圆内，一根在外。记 z1=−a+a2−1z_1 = -a + \sqrt{a^2-1}z1​=−a+a2−1​（绝对值小于 111），则

I=2i⋅2πi Res[1z2+2az+1,z1]=4π⋅12z1+2a\begin{align} I = \frac{2}{\mathrm{i}} \cdot 2\pi \mathrm{i} \, \mathrm{Res}\left[ \frac{1}{z^2+2a z+1}, z_1 \right] = 4\pi \cdot \frac{1}{2z_1+2a} \end{align}I=i2​⋅2πiRes[z2+2az+11​,z1​]=4π⋅2z1​+2a1​​​

例2：计算一个有理函数的无穷积分

计算 I=∫−∞∞dx(x2+1)(x2+4)I = \int_{-\infty}^{\infty} \frac{dx}{(x^2+1)(x^2+4)}I=∫−∞∞​(x2+1)(x2+4)dx​。

解：被积函数是有理函数，分母次数比分子高 444，满足类型二条件。上半平面内的奇点为 z=iz = \mathrm{i}z=i（一阶极点）和 z=2iz = 2\mathrm{i}z=2i（一阶极点？注意分母 (z2+1)(z2+4)=(z−i)(z+i)(z−2i)(z+2i)(z^2+1)(z^2+4) = (z-\mathrm{i})(z+\mathrm{i})(z-2\mathrm{i})(z+2\mathrm{i})(z2+1)(z2+4)=(z−i)(z+i)(z−2i)(z+2i)，所以 z=i,2iz=\mathrm{i},2\mathrm{i}z=i,2i 均为单极点）。计算留数：

Res[1(z2+1)(z2+4),i]=lim⁡z→iz−i(z−i)(z+i)(z2+4)=1(2i)(−1+4)=16i\begin{align} \mathrm{Res}\left[ \frac{1}{(z^2+1)(z^2+4)}, \mathrm{i} \right] = \lim_{z\to \mathrm{i}} \frac{z-\mathrm{i}}{(z-\mathrm{i})(z+\mathrm{i})(z^2+4)} = \frac{1}{(2\mathrm{i})(-1+4)} = \frac{1}{6\mathrm{i}} \end{align}Res[(z2+1)(z2+4)1​,i]=z→ilim​(z−i)(z+i)(z2+4)z−i​=(2i)(−1+4)1​=6i1​​​ Res[1(z2+1)(z2+4),2i]=lim⁡z→2iz−2i(z2+1)(z−2i)(z+2i)=1((−4)+1)(4i)=1−3⋅4i=−112i\begin{align} \mathrm{Res}\left[ \frac{1}{(z^2+1)(z^2+4)}, 2\mathrm{i} \right] = \lim_{z\to 2\mathrm{i}} \frac{z-2\mathrm{i}}{(z^2+1)(z-2\mathrm{i})(z+2\mathrm{i})} = \frac{1}{((-4)+1)(4\mathrm{i})} = \frac{1}{-3\cdot 4\mathrm{i}} = -\frac{1}{12\mathrm{i}} \end{align}Res[(z2+1)(z2+4)1​,2i]=z→2ilim​(z2+1)(z−2i)(z+2i)z−2i​=((−4)+1)(4i)1​=−3⋅4i1​=−12i1​​​

例3：计算一个含三角函数的无穷积分

计算 I=∫−∞∞cos⁡xx2+1 dxI = \int_{-\infty}^{\infty} \frac{\cos x}{x^2 + 1} \, dxI=∫−∞∞​x2+1cosx​dx。

Res[eizz2+1,i]=ei⋅i2i=e−12i\begin{align} \mathrm{Res}\left[ \frac{e^{\mathrm{i} z}}{z^2+1}, \mathrm{i} \right] = \frac{e^{\mathrm{i} \cdot \mathrm{i}}}{2\mathrm{i}} = \frac{e^{-1}}{2\mathrm{i}} \end{align}Res[z2+1eiz​,i]=2iei⋅i​=2ie−1​​​

由留数定理，

J=2πi⋅e−12i=πe−1\begin{align} J = 2\pi \mathrm{i} \cdot \frac{e^{-1}}{2\mathrm{i}} = \pi e^{-1} \end{align}J=2πi⋅2ie−1​=πe−1​​
