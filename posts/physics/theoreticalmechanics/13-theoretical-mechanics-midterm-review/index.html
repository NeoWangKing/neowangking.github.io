一、牛顿力学基础

牛顿三定律与常见坐标系

  • 第一定律：惯性参考系中物体保持静止或匀速直线运动。
  • 第二定律：F=dp/dt\boldsymbol{F} = \mathrm{d}\boldsymbol{p}/\mathrm{d}tF=dp/dt，质量恒定则为 F=ma\boldsymbol{F}=m\boldsymbol{a}F=ma。
  • 第三定律：F12=−F21\boldsymbol{F}_{12} = -\boldsymbol{F}_{21}F12​=−F21​。

在不同坐标系下，加速度分量需要熟悉：

  • 平面极坐标：ar=r¨−rθ˙2a_r = \ddot{r} - r\dot{\theta}^2ar​=r¨−rθ˙2，aθ=rθ¨+2r˙θ˙a_\theta = r\ddot{\theta} + 2\dot{r}\dot{\theta}aθ​=rθ¨+2r˙θ˙
  • 自然坐标：at=s¨a_t = \ddot{s}at​=s¨，an=v2/ρa_n = v^2/\rhoan​=v2/ρ

质点系与守恒定律

  • 质心：rc=∑miri/M\boldsymbol{r}_c = \sum m_i \boldsymbol{r}_i / Mrc​=∑mi​ri​/M，质心运动定理：Mr¨c=F(ext)M\ddot{\boldsymbol{r}}_c = \boldsymbol{F}^{\text{(ext)}}Mr¨c​=F(ext)
  • 动量定理：dP/dt=F(ext)\mathrm{d}\boldsymbol{P}/\mathrm{d}t = \boldsymbol{F}^{\text{(ext)}}dP/dt=F(ext)
  • 角动量定理：dL/dt=M(ext)\mathrm{d}\boldsymbol{L}/\mathrm{d}t = \boldsymbol{M}^{\text{(ext)}}dL/dt=M(ext)，有心力下角动量守恒
  • 动能定理：dT=F⋅dr\mathrm{d}T = \boldsymbol{F}\cdot \mathrm{d}\boldsymbol{r}dT=F⋅dr，保守力下机械能守恒

二、分析力学基础

约束与广义坐标

约束：对质点系运动的几何或运动学限制。

  • 完整约束：只含坐标（和时间）的等式约束 f(ri,t)=0f(\boldsymbol{r}_i,t)=0f(ri​,t)=0，或可积分的微分约束。
  • 非完整约束：含有速度且不可积分。
  • 自由度：完整体系中独立坐标个数 s=3n−ks = 3n - ks=3n−k（nnn 质点数，kkk 完整约束数）。

广义坐标 q1,…,qsq_1,\dots,q_sq1​,…,qs​：任意一组完全确定系统位形的独立参数。

虚位移与虚功

  • 虚位移 δr\delta\boldsymbol{r}δr：满足瞬时约束的无限小位移，与时间变化无关。

  • 虚功：δW=F⋅δr\delta W = \boldsymbol{F}\cdot\delta\boldsymbol{r}δW=F⋅δr。

  • 理想约束：约束反力在任意虚位移上总虚功为零：

    ∑iNi⋅δri=0\begin{align} \sum_i \boldsymbol{N}_i \cdot \delta\boldsymbol{r}_i = 0 \end{align}i∑​Ni​⋅δri​=0​​

    光滑曲面、无质量刚性杆、纯滚动、不可伸长绳等均属理想约束。

达朗贝尔方程

从牛顿第二定律出发，引入惯性力，在理想约束下得到不含约束力的动力学方程：

∑i(Fi−mir¨i)⋅δri=0\begin{align} \sum_i (\boldsymbol{F}_i - m_i \ddot{\boldsymbol{r}}_i) \cdot \delta\boldsymbol{r}_i = 0 \end{align}i∑​(Fi​−mi​r¨i​)⋅δri​=0​​

静力学特例 → 虚功原理：∑iFi⋅δri=0\sum_i \boldsymbol{F}_i \cdot \delta\boldsymbol{r}_i = 0∑i​Fi​⋅δri​=0。

三、拉格朗日方程

一般形式

对于 理想、完整体系，定义广义力

Qα=∑iFi⋅∂ri∂qα\begin{align} Q_\alpha = \sum_i \boldsymbol{F}_i \cdot \frac{\partial \boldsymbol{r}_i}{\partial q_\alpha} \end{align}Qα​=i∑​Fi​⋅∂qα​∂ri​​​​

拉格朗日方程为

ddt(∂T∂q˙α)−∂T∂qα=Qα,α=1,…,s\begin{align} \frac{\mathrm{d}}{\mathrm{d}t}\left(\frac{\partial T}{\partial \dot{q}_\alpha}\right) - \frac{\partial T}{\partial q_\alpha} = Q_\alpha, \quad \alpha=1,\dots,s \end{align}dtd​(∂q˙​α​∂T​)−∂qα​∂T​=Qα​,α=1,…,s​​

保守体系的拉格朗日方程

若主动力为保守力，Fi=−∇iV\boldsymbol{F}_i = -\nabla_i VFi​=−∇i​V，引入拉格朗日函数 L=T−VL = T - VL=T−V，有

ddt(∂L∂q˙α)−∂L∂qα=0\begin{align} \frac{\mathrm{d}}{\mathrm{d}t}\left(\frac{\partial L}{\partial \dot{q}_\alpha}\right) - \frac{\partial L}{\partial q_\alpha} = 0 \end{align}dtd​(∂q˙​α​∂L​)−∂qα​∂L​=0​​

此形式与坐标选择无关（协变性），且不含约束力。

解题步骤

  1. 判断约束类型，确定自由度 sss。
  2. 选取广义坐标 qαq_\alphaqα​。
  3. 写出动能 T(q,q˙,t)T(q,\dot{q},t)T(q,q˙​,t) 和势能 V(q,t)V(q,t)V(q,t)，构造 L=T−VL = T - VL=T−V。
  4. 若有非保守力，计算广义力。
  5. 代入拉格朗日方程，得到 sss 个二阶常微分方程。

四、平衡问题与广义力

平衡条件

由虚功原理，平衡时广义力为零：

Qα=0,α=1,…,s\begin{align} Q_\alpha = 0, \quad \alpha=1,\dots,s \end{align}Qα​=0,α=1,…,s​​

对保守体系等价于 ∂V/∂qα=0\partial V / \partial q_\alpha = 0∂V/∂qα​=0，即势能取极值。

  • 稳定平衡：势能取极小值，二阶偏导数矩阵正定。
  • 不稳定平衡：势能取极大值或鞍点。

利用平衡求约束力（释放约束法）

  1. 解除欲求约束力对应的约束。
  2. 将约束力视为主动力，在相应虚位移上做功。
  3. 平衡条件 δW=0\delta W = 0δW=0 求解该力。

五、对称性与守恒定律

运动积分

运动积分是坐标、速度和时间的函数，在运动过程中保持常数。

广义能量积分

若 LLL 不显含时间（∂L/∂t=0\partial L / \partial t = 0∂L/∂t=0），则

H=∑α∂L∂q˙αq˙α−L=常数\begin{align} H = \sum_\alpha \frac{\partial L}{\partial \dot{q}_\alpha} \dot{q}_\alpha - L = \text{常数} \end{align}H=α∑​∂q˙​α​∂L​q˙​α​−L=常数​​
  • 稳定约束下，动能是广义速度的二次齐次函数，H=T+VH = T + VH=T+V 即机械能。
  • 否则 HHH 只是广义能量，不一定是机械能。

循环坐标与广义动量守恒

若 LLL 不显含某坐标 qαq_\alphaqα​（循环坐标），则对应的广义动量守恒：

pα=∂L∂q˙α=常数\begin{align} p_\alpha = \frac{\partial L}{\partial \dot{q}_\alpha} = \text{常数} \end{align}pα​=∂q˙​α​∂L​=常数​​

诺特定理

连续对称性必然导出守恒量：

  • 空间平移对称性 → 动量守恒
  • 空间旋转对称性 → 角动量守恒
  • 时间平移对称性 → 广义能量守恒

九、多自由度体系的微振动
