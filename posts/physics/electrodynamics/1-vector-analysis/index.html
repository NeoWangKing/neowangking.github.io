矢量代数

基本运算

  1. 加法：A+B=B+A\boldsymbol{A} + \boldsymbol{B} = \boldsymbol{B} + \boldsymbol{A}A+B=B+A，满足交换律和结合律。
  2. 数乘：α(A+B)=αA+αB\alpha(\boldsymbol{A}+\boldsymbol{B}) = \alpha\boldsymbol{A} + \alpha\boldsymbol{B}α(A+B)=αA+αB。
  3. 点乘（标量积）：A⋅B=ABcos⁡θ\boldsymbol{A} \cdot \boldsymbol{B} = AB\cos\thetaA⋅B=ABcosθ，满足交换律 A⋅B=B⋅A\boldsymbol{A}\cdot\boldsymbol{B} = \boldsymbol{B}\cdot\boldsymbol{A}A⋅B=B⋅A 和分配律 A⋅(B+C)=A⋅B+A⋅C\boldsymbol{A}\cdot(\boldsymbol{B}+\boldsymbol{C}) = \boldsymbol{A}\cdot\boldsymbol{B} + \boldsymbol{A}\cdot\boldsymbol{C}A⋅(B+C)=A⋅B+A⋅C。
  4. 叉乘（矢量积）：A×B=ABsin⁡θ n^\boldsymbol{A} \times \boldsymbol{B} = AB\sin\theta\,\hat{\boldsymbol{n}}A×B=ABsinθn^，其中 n^\hat{\boldsymbol{n}}n^ 是垂直于 A,B\boldsymbol{A},\boldsymbol{B}A,B 所在平面且按右手定则确定的单位矢量。叉乘满足分配律，但不满足交换律：B×A=−(A×B)\boldsymbol{B}\times\boldsymbol{A} = -(\boldsymbol{A}\times\boldsymbol{B})B×A=−(A×B)，且 A×A=0\boldsymbol{A}\times\boldsymbol{A}=0A×A=0。

三重积

  • 标量三重积（混合积）：

    A⋅(B×C)=B⋅(C×A)=C⋅(A×B)\begin{align} \boldsymbol{A}\cdot(\boldsymbol{B}\times\boldsymbol{C}) = \boldsymbol{B}\cdot(\boldsymbol{C}\times\boldsymbol{A}) = \boldsymbol{C}\cdot(\boldsymbol{A}\times\boldsymbol{B}) \end{align}A⋅(B×C)=B⋅(C×A)=C⋅(A×B)​​

    其几何意义是以三个矢量为棱的平行六面体的体积。

  • 矢量三重积：

    A×(B×C)=B(A⋅C)−C(A⋅B)\begin{align} \boldsymbol{A}\times(\boldsymbol{B}\times\boldsymbol{C}) = \boldsymbol{B}(\boldsymbol{A}\cdot\boldsymbol{C}) - \boldsymbol{C}(\boldsymbol{A}\cdot\boldsymbol{B}) \end{align}A×(B×C)=B(A⋅C)−C(A⋅B)​​

    此公式称为“BAC-CAB”规则，需注意括号位置。

  • 四个矢量的点乘关系：

    (A×B)⋅(C×D)=(A⋅C)(B⋅D)−(A⋅D)(B⋅C)\begin{align} (\boldsymbol{A}\times\boldsymbol{B})\cdot(\boldsymbol{C}\times\boldsymbol{D}) = (\boldsymbol{A}\cdot\boldsymbol{C})(\boldsymbol{B}\cdot\boldsymbol{D}) - (\boldsymbol{A}\cdot\boldsymbol{D})(\boldsymbol{B}\cdot\boldsymbol{C}) \end{align}(A×B)⋅(C×D)=(A⋅C)(B⋅D)−(A⋅D)(B⋅C)​​

梯度

定义：标量场 T(x,y,z)T(x,y,z)T(x,y,z) 的梯度是一个矢量，表示 TTT 在空间变化最快的方向与速率：

散度

定义：矢量场 v\boldsymbol{v}v 的散度是一个标量，衡量场从某一点发散的程度：

∇⋅v=∂vx∂x+∂vy∂y+∂vz∂z\begin{align} \nabla\cdot\boldsymbol{v} = \frac{\partial v_x}{\partial x} + \frac{\partial v_y}{\partial y} + \frac{\partial v_z}{\partial z} \end{align}∇⋅v=∂x∂vx​​+∂y∂vy​​+∂z∂vz​​​​

几何意义：旋度方向为最大环量密度的方向，大小反映旋转的强弱。

矢量的积分学

梯度定理（基本定理）

对于梯度场，其路径积分与路径无关，仅取决于端点：

∫ab(∇T)⋅dl=T(b)−T(a)\begin{align} \int_{\boldsymbol{a}}^{\boldsymbol{b}} (\nabla T)\cdot d\boldsymbol{l} = T(\boldsymbol{b}) - T(\boldsymbol{a}) \end{align}∫ab​(∇T)⋅dl=T(b)−T(a)​​

推论：沿闭合回路积分 ∮(∇T)⋅dl=0\oint (\nabla T)\cdot d\boldsymbol{l}=0∮(∇T)⋅dl=0。

散度定理（高斯定理）

矢量场通过闭合曲面的通量等于其散度在曲面所围体积内的积分：

∭V(∇⋅v) dτ=∯S=∂Vv⋅da\begin{align} \iiint_{V} (\nabla\cdot\boldsymbol{v})\,d\tau = \oiint_{S=\partial V} \boldsymbol{v}\cdot d\boldsymbol{a} \end{align}∭V​(∇⋅v)dτ=∬​S=∂V​v⋅da​​

旋度定理（斯托克斯定理）

矢量场在曲面上的通量（旋度的面积分）等于其沿曲面边界的环量：

∬S(∇×v)⋅da=∮L=∂Sv⋅dl\begin{align} \iint_{S} (\nabla\times\boldsymbol{v})\cdot d\boldsymbol{a} = \oint_{L=\partial S} \boldsymbol{v}\cdot d\boldsymbol{l} \end{align}∬S​(∇×v)⋅da=∮L=∂S​v⋅dl​​

推论：对于闭合曲面，∮(∇×v)⋅da=0\oint (\nabla\times\boldsymbol{v})\cdot d\boldsymbol{a}=0∮(∇×v)⋅da=0。

其中 ei\boldsymbol{e}_iei​ 是基矢量，eiej\boldsymbol{e}_i\boldsymbol{e}_jei​ej​ 表示并矢。

在连续介质中，作用在某面元上的力 F\boldsymbol{F}F 与该面元的定向有关，可用应力张量 T\boldsymbol{T}T 描述：

dF=T⋅da\begin{align} d\boldsymbol{F} = \boldsymbol{T}\cdot d\boldsymbol{a} \end{align}dF=T⋅da​​

分量 TijT_{ij}Tij​ 表示在垂直于 jjj 方向的单位面积上沿 iii 方向的力。

张量的代数运算

  • 数乘与加法：(αT)ij=αTij(\alpha\boldsymbol{T})_{ij} = \alpha T_{ij}(αT)ij​=αTij​，(T+P)ij=Tij+Pij(\boldsymbol{T}+\boldsymbol{P})_{ij} = T_{ij}+P_{ij}(T+P)ij​=Tij​+Pij​。
  • 张量与矢量的点乘（左乘和右乘结果不同）： (v⋅T)i=∑jvjTji,(T⋅v)i=∑jvjTij\begin{align} (\boldsymbol{v}\cdot\boldsymbol{T})_i = \sum_j v_j T_{ji}, \quad (\boldsymbol{T}\cdot\boldsymbol{v})_i = \sum_j v_j T_{ij} \end{align}(v⋅T)i​=j∑​vj​Tji​,(T⋅v)i​=j∑​vj​Tij​​​ 两者一般不相等。
  • 张量间的点乘：(T⋅P)ij=∑kTikPkj(\boldsymbol{T}\cdot\boldsymbol{P})_{ij} = \sum_k T_{ik} P_{kj}(T⋅P)ij​=∑k​Tik​Pkj​。
  • 散度：张量场的散度定义为矢量，其第 iii 分量为 (∇⋅T)i=∑j∂Tji∂xj\begin{align} (\nabla\cdot\boldsymbol{T})_i = \sum_j \frac{\partial T_{ji}}{\partial x_j} \end{align}(∇⋅T)i​=j∑​∂xj​∂Tji​​​​

张量的坐标变换

设坐标变换矩阵为 R\mathsf{R}R（正交矩阵），则矢量变换为 A~i=∑jRijAj\tilde{A}_i = \sum_j R_{ij}A_jA~i​=∑j​Rij​Aj​。二阶张量变换为：

T~ij=∑k,lRikRjlTkl\begin{align} \tilde{T}_{ij} = \sum_{k,l} R_{ik} R_{jl} T_{kl} \end{align}T~ij​=k,l∑​Rik​Rjl​Tkl​​​

即矩阵形式 T~=RTRT\tilde{\mathsf{T}} = \mathsf{R}\mathsf{T}\mathsf{R}^{\mathsf{T}}T~=RTRT。
