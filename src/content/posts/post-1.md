## C语言程序结构

C语言主要包括以下部分：

- 预处理指令：告诉编译器在正式编译之前要进行的操作（比如要包含某个文件）
- 函数
- 变量
- 语句 & 表达式
- 注释

## C语言基础语法

C语言的Token有几种类型：

- 关键字（Keywords）
- 标识符（Identifiers）
- 常量（Constants）
- 字符串字面量（String Literals）
- 运算符（Operators）
- 分隔符（Separators）

示例(code2.c)

```
#include <stdio.h>   // 头文件包含

#define PI 3.14159    // 宏定义

// 函数声明
int add(int a, int b);

int main() {         // 主函数
    // 变量声明
    int num1, num2, sum;

    // 用户输入
    printf("Enter two integers: ");
    scanf("%d %d", &num1, &num2);

    // 函数调用
    sum = add(num1, num2);

    // 输出结果
    printf("Sum: %d\n", sum);

    return 0;        // 返回 0 表示程序成功执行
}

// 函数定义
int add(int a, int b) {
    return a + b;
}
```

> - 头文件包含：在程序的开头使用`#include`指令包含头文件（例如：标准输入输出库`<stdio.h>`、标准库`<stdlib.h>`）
> - 宏定义：通过`#define`指令定义的符号常量或代码片段，在编译前由预处理器替换为其定义的内容（例如：`#define PI 3.14159`）
> - 函数声明：函数的声明必须在函数定义或调用之前，声明了函数的返回类型、函数名、参数列表
> - 主函数：`main()`函数是C语言程序的入口，每个C程序都必须包含一个main函数，通常把返回值设置成0
> - 变量声明：C程序中所有变量必须在使用前声明
> - 语句：语句是程序的基本执行单元，通常是函数调用、赋值、控制语句
> - 表达式：由变量、操作符、常量组成的代码片段
> - 控制流语句：用于控制程序执行的顺序（如`if`、`for`、`while`、`do-while`语句）
> - 函数定义：包含实际的函数体，包含参数、局部变量和返回值
> - 返回语句：`return`语句用于终止函数的执行，并将控制权交还给调用函数

### 分隔符

- 逗号`,`：用于分割变量声明或函数参数
- 分号`;`：用于结束语句
- 括号：
  - 圆括号`()`：用于分组表达式、函数调用
  - 花括号`{}`：用于定义代码块
  - 方括号`[]`：用于数组下标

### 注释

- 单行注释：`// 单行注释`
- 多行注释：`/* ... */`

### 标识符

程序中变量、函数、数组等的名字  
标识符由字母（大写或小写）、数字和下划线组成，但第一个字符必须是字母或下划线，不能是数字

### 常量

常量是在程序执行期间不会改变的固定值，用`const`命令声明

### 字符串字面量

由双引号括起来的字符序列，末尾会自动添加一个空字符`\0`

### 运算符（Operators）

- 算数运算符：`+`,`-`,`*`,`/`,`%`
- 关系运算符：`==`,`!=`,`>`,`<`,`>=`,`<=`
- 逻辑运算符：`&&`,`||`,`!`
- 位运算符：`&`,`|`,`` ` ``,`~`,`<<`,`>>`
- 赋值运算符：`=`,`+=`,`-=`,`*=`,`/=`,`%=`
- 其他运算符：`sizeof`,`?:`,`&`,`*`,`->`,`.`……

## C数据类型

| 序号 | 类型与描述                                                                     |
| :--- | :----------------------------------------------------------------------------- |
| 1    | 基本数据类型：是算术类型，包括整型、字符型、浮点型、双精度浮点型               |
| 2    | 枚举类型：它们也是算术类型，被用来定义在程序中只能赋予其一定的离散整数值的变量 |
| 3    | void类型：类型说明符 void 表示没有值的数据类型，通常用于函数返回值             |
| 4    | 派生类型：包括数组类型、指针类型和结构体类型                                   |
